#Product Management System - CodeIgniter 4

Aplikasi ini adalah sistem manajemen produk berbasis CodeIgniter 4. Fitur utamanya meliputi login user, manajemen produk (tambah, edit, hapus, lihat daftar), dan manajemen kategori.

Aplikasi menggunakan PHP 8, CodeIgniter 4, MySQL, dan Bootstrap 5 untuk tampilan.

#Fitur

Login user dengan email dan password.

Tambah, edit, hapus, dan lihat daftar produk.

Lihat daftar kategori produk.

API endpoint untuk produk dan kategori.

#Instalasi

Clone repository ke komputer, install dependencies dengan composer, dan konfigurasi database di file .env. Jalankan migration untuk membuat tabel, lalu jalankan server. Akses aplikasi melalui browser di alamat localhost.

#Penggunaan

Buka halaman login, masukkan email dan password, lalu masuk. Setelah login, buka halaman produk untuk melihat daftar produk, menambah produk baru, mengedit, atau menghapus produk. Kategori akan otomatis muncul saat menambah atau mengedit produk.

#Catatan

Pastikan database sudah dibuat dan terhubung dengan benar. Gunakan password yang aman. Bootstrap 5 sudah digunakan untuk tampilan. Endpoint API bisa diuji dengan Postman jika diperlukan.
